# Color Palette

## Base Colors

| Name          | Hex       | Usage                        |
|---------------|-----------|------------------------------|
| bg-dark       | `#0a0a0a` | Primary background           |
| bg            | `#121210` | Secondary background         |
| bg-light      | `#1a1916` | Elevated surfaces            |
| bg-lighter    | `#252420` | Borders, subtle highlights   |
| surface       | `#2e2c26` | Cards, panels                |
| overlay       | `#3a3730` | Overlays, popups             |

## Foreground Colors

| Name          | Hex       | Usage                        |
|---------------|-----------|------------------------------|
| fg            | `#d4c9a8` | Primary text                 |
| fg-muted      | `#a89f84` | Secondary text, comments     |
| fg-subtle     | `#6b6454` | Disabled, placeholders       |
| fg-bright     | `#e8dfc0` | Emphasized text              |

## Accent Colors

| Name          | Hex       | Usage                        |
|---------------|-----------|------------------------------|
| amber         | `#c9a227` | Primary accent               |
| gold          | `#d4a84b` | Highlights, links            |
| ochre         | `#b8860b` | Warnings, secondary accent   |
| olive         | `#6b7353` | Success, strings             |
| teal          | `#4a6b5a` | Info, types                  |
| rust          | `#8b4a2b` | Errors, deletions            |
| sepia         | `#8b7355` | Special, tags                |
| bronze        | `#a67c52` | Functions, methods           |

## Quick Reference

```
Background:  #0a0a0a / #121210 / #1a1916
Foreground:  #d4c9a8 / #a89f84
Primary:     #c9a227 (amber)
Secondary:   #d4a84b (gold)
Accent:      #6b7353 (olive) / #4a6b5a (teal)
Error:       #a64a3a
```
